# Scratch

Ebben a projektben egy boszorkányt irányítunk, amivel tűzgolyókat kell lőnünk a szembejövő seprűkre.
Ha hozzáérünk egy seprűhöz, akkor a játéknak vége.

Mozgás: 'Le' és 'Fel' nyilak, lövés: Space.
